# Phase 3: Dependencies Installation

## Required Package

Install recharts for analytics charts:

```cmd
cd C:\Users\admin\Projects\resume-analyzer\frontend
npm install recharts
```

Then commit:

```cmd
cd ..
git add frontend/package.json frontend/package-lock.json
git commit -m "Add recharts for analytics charts"
git push
```

After installation, the analytics features will be fully functional.
